About me...

I'm Front-end Engineer and I live at Campinas/SP. I have bachelor degree of Information System and I've been working over more than eight years with Javascript, HTML, CSS, Java, C# e NodeJS.

At free time, I like to read and learn about UX, Javascript and Technology in general, where my goal is improve my knowledge about the amazing world of web development.

What I do

I know a lot of cool things, but I'm always looking for innovation!

Back-end

I usually build Web Applications with Java, C# or NodeJS platforms, which in general provide RESTful APIs. I always try to use industry's best practices to deliver scalable solutions and full of quality.

Front-end

I use Javascript as my current platform to build front-end solutions. AngularJS is where I feel confortable to work with, but I love to create my own libraries with vanilla Javascript and also try things with React and VueJS.

Workflow

More than never is possible keep a productive workflow, thanks to all task runners and package managers out there on the web. I've experienced Gulp, NPM, Webpack, Browserify, and I enjoy anyone which can provide what I need, depending the requirements of each project I'm working on.

Versioning

I use Git as my version control system since I got started with web development. It's trustable and full of features that grant the integrity of my code.
